Welcome to Notiforward

Notiforward is an open-source notification forwarding platform that uses your own database that you control. The platform currently consists of an Android app and Chrome extension. The app and extension communicate with each other using a Firebase database that only you have access too.

Getting Started

  1. Creating your own database is very simple and takes only a few minutes. You can follow our step by step guide.
  2. After setting up your database, you can download both the Chrome extension and Android app.
    • Download Chrome Extension
    • Download Android App
